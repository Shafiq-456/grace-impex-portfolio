# Grace Impex — Premium Textile Machinery Import

A lightweight portfolio / landing site for Grace Impex showcasing imported textile and garment machinery.

## Features
- Simple, responsive single-page front-end (React + Vite / TypeScript assumed)
- SEO meta tags and Open Graph / Twitter preview image (`public/preview.png`)
- Favicon (`public/favicon.ico`)
- Easy local development and build scripts

## Preview
Place `preview.png` and `favicon.ico` in the `public/` folder. Social previews and browser favicon will use these assets.

## Requirements
- Node.js (16+) and npm or yarn
- Git (for cloning / contributing)

## Local development
```bash
# from project root
npm install
npm run dev        # start dev server (Vite)
```

## Build
```bash
npm run build      # build production assets
npm run preview    # locally preview production build (optional)
```

## Deployment
Deploy the contents of the `dist` (or `build`) folder to your static host (Netlify, Vercel, GitHub Pages, etc.). If using GitHub Pages, configure the build step in your CI or use the `gh-pages` package.
